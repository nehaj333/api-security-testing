# API Security Testing

This repository demonstrates common API security vulnerabilities and testing techniques.

## API Security Risks

- Broken Authentication
- IDOR
- Rate Limit Bypass
- Sensitive Data Exposure
- JWT Manipulation

## Tools Used

- Burp Suite
- Postman
- OWASP ZAP

## Testing Methodology

1. Intercept API requests
2. Modify parameters
3. Test authorization checks
4. Attempt privilege escalation# api-security-testing
